# CS-Lite Web v0.1

A browser-based multiplayer 3D skeleton.

## v0.1 Scope

- Two real players
- Public room join by room code
- Simple Babylon.js test map
- WASD + mouse look
- Remote player visibility
- Colyseus state sync
- Render Free deployment

## Not Included in v0.1

- Shooting
- AI
- Rounds
- Scoring
- 5v5
- Database
- Accounts

## Local Development

```bash
npm install
npm run dev
```

Client:

```text
http://localhost:5173
```

Server:

```text
http://localhost:2567
```

## Build

```bash
npm run build
```
